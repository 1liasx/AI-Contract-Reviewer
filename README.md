# Sample data and test cases

All names, companies, addresses, dates, and amounts in this folder are fictional and are provided only for testing.

| File | Main condition | Expected risk |
| --- | --- | --- |
| `01-low-risk-contract.pdf` | Balanced terms, clear scope and termination | `LOW` |
| `02-medium-risk-contract.pdf` | Missing service credits and ambiguous renewal terms | `MEDIUM` |
| `03-high-risk-contract.pdf` | Unlimited liability, unilateral changes, and no termination right | `HIGH` |
| `04-invalid-file.txt` | Unsupported MIME type | HTTP `400` |

Because the result is generated by an LLM, exact wording and confidence values are non-deterministic. Treat `expected-results.json` as validation criteria, not as a byte-for-byte snapshot.

Send a PDF using the multipart field name `file`:

```bash
curl -X POST "https://YOUR_N8N_DOMAIN/webhook/contract-review" \
  -F "file=@01-low-risk-contract.pdf;type=application/pdf"
```
